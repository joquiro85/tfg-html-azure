Recursos Azure

Zona DNS

Apunta el registro A del dominio raíz al Load Balancer y crea un CNAME para que www.tfg-joqr.es redirija al dominio raíz.

Resource Group

Data source para referenciar el grupo de recursos TFG-Infra.

Load Balancer

Load Balancer Standard que reparte el tráfico HTTP en el puerto 80 entre las instancias backend y dirige las conexiones TCP/80 desde el frontend al pool.

Jumpbox VM

VM Linux para SSH y administración en la subred pública.

Autoscale Setting

Escala automáticamente el número de instancias en función de la carga de CPU.

NAT Gateway

Proporciona salida a Internet para el subnet privado.Vincula la IP pública al NAT Gateway.

Network Security Groups

Asocia el Security Group de MySQL a su interfaz y define reglas para permitir MySQL (3306) desde la subred pública, así como HTTP y SSH entrantes al jumpbox y al VMSS.

IP Públicas

Proporciona IPs estáticas para tres funciones: el frontend del Load Balancer, el tráfico saliente del NAT Gateway y el acceso SSH al jumpbox.

Private Subnet

Subnet privado asociado al NAT Gateway. instancias del VMSS y MySQL.

Public Subnet

Subnet público para el jumpbox y LB.

Virtual Network

Red virtual que engloba todas las subnets.
