Recursos

Amazon EC2

Instancias EC2 tanto públicas (nginx) como privadas (MySQL) como parte del backend del proyecto. Las instancias se gestionan con ASG y SSM.

VPC y Subredes

Red VPC personalizada con subredes públicas y privadas distribuidas en dos zonas de disponibilidad para segmentación y seguridad.

NAT Gateway

Permite que las instancias privadas accedan a Internet sin recibir conexiones entrantes.

Security Groups

Grupos de seguridad para controlar el tráfico en EC2, ALB y MySQL. Separan roles y protegen los accesos entre recursos.

Launch Template

Define los parámetros predeterminados de instancias EC2, incluyendo AMI, tipo, clave, SSM y user_data.sh.

IAM Role & Instance Profile

Permite acceso SSM sin claves SSH usando el rol AmazonSSMManagedInstanceCore.

Tablas de Ruteo

Tablas para enrutar tráfico desde y hacia Internet mediante IGW y NAT Gateway.

Internet Gateway

Permite que recursos en subredes públicas se comuniquen con Internet.

Route 53

Asignación de dominio tfg-joqr.es al Load Balancer mediante registros A y CNAME.

Auto Scaling Group

Escalado automático de instancias NGINX en subredes privadas, con mínimo 2 y máximo 5 instancias.

Application Load Balancer

Balanceador HTTP en subredes públicas conectado a Target Group de instancias privadas.
